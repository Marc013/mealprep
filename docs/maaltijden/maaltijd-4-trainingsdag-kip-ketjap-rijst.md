## Ingrediënten per portie

| Ingrediënt        | Hoeveelheid |
| :---------------- | ----------: |
| Kipfilet          |       140 g |
| Witte rijst droog |        80 g |
| Champignons       |       100 g |
| Paprika           |       100 g |
| Ui                |        80 g |
| Courgette         |       150 g |
| Taugé             |        75 g |
| Ketjap manis      |       15 ml |
| Sojasaus          |        5 ml |
| Olijfolie         |        3 ml |
| Knoflook          |   1 teentje |

## Bereiding

<ol class="cooking-steps">
  <li>Kook de rijst gaar volgens de verpakking.</li>
  <li>Bak de kip op hoog vuur goudbruin.</li>
  <li>Voeg ui, knoflook, paprika, champignons en courgette toe en roerbak stevig.</li>
  <li>Voeg taugé, ketjap en sojasaus toe en bak nog kort door.</li>
  <li>Verdeel samen met de rijst over mealprep-bakken.</li>
</ol>

## Praktisch

- Verwarm 3 tot 4 minuten op 700W en roer halverwege door.
- Voor de exacte zondag- en woensdagbatch: zie [Batchprep 1 Week]({{ '/batchprep-1-week' | relative_url }}).
